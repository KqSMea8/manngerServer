属性

含义

默认值

subGrid

是否使用subgrid模式

false

subGridOptions

subgrid属性

{

plusicon : “ui-icon-plus”,
minusicon : “ui-icon-minus”,
openicon: “ui-icon-carat-1-sw”,
expandOnLoad: false,
selectOnExpand : false,
reloadOnExpand : true

}

plusicon：展开subGrid图标；

minusicon：折叠subGrid图标；

openicon：展开subGrid时minusicon下面内容图标；

expandOnLoad：是否自动展开所有行；

selectOnExpand：是否点击展开图标时当前行被选择；

reloadOnExpand：是否subgrid内容仅加载一次

subGridModel

subgrid模型定义

[

{ name : ['name_1','name_2',…,'name_n'],
width : [width_1,width_2,…,width_n] ,
align : ['left','center',…,'right'] ,
params : [param_1,…,param_n],
mapping:['name_1_map','name_2_map',…]}]

name：数组，包含subgrid的列标题；

width：数组，包列宽度；

align：数组，包含列对齐方式，值可为left、center、right；

params：数组，额外参数；

mapping：当repeatitems设置为false时，在一个应用中把数组中names映射到subgrid的names。

subgridtype

设置次网格数据类型，如不设置使用父grid的datatype类型

null

subGridWidth

次网格列宽度

20

subGridUrl

获取Subgrid数据源，附加行id将作为参数传递，如需更多参数请在subGridModel中设置

“”
