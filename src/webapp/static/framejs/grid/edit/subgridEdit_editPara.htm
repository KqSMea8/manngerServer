事件

afterComplete

新增或者更新数据完成后触发

 

response：server端返回数据；

postdata：发送到server端的数据；

formid：弹出框中form id值；

beforeCheckValues

校验单元格值前触发（colModel中editrules定义的规则）

postdata：发送到server端的数据；

formid：弹出框中form id值；

mode：当前模型（编辑或新增）

beforeInitData

初始化dialog新数据前触发

formid：弹出框中form id值；

beforeSubmit

提交数据到server前触发

postdata：发送到server端的数据；

formid：弹出框中form id值；

此事件要求返回数据格式为：

[success, message]

success：为true时继续处理，为false时显示message信息并停止后续处理。

onclickSubmit

点击“提交”按钮时触发

Params：editGridRow方法所有属性；

postdata：发送到server端的数据；

此事件要求返回数据格式为：

{}，扩展postdata；

onInitializeForm

首次创建dialog时触发

formid：弹出框中form id值；

errorTextFormat

Ajax请求错误时触发

返回错误信息

serializeEditData

提交数据时序列化提交到server端数据

可以设置自定义数据
