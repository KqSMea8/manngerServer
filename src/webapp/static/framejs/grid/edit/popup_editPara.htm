说明

基本属性

top

0

弹出框top位置（相对grid位置）

left

0

弹出框left位置（相对grid位置）

width

300

宽度

height

auto

数据区域高度

dataheight

auto

弹出框可缩放大小

modal

false

是否弹窗口为模态窗口

drag

true

是否可拖拽

resize

true

是否可调整大小

url

null

提交数据地址，如设置覆盖editurl

mtype

POST

提交数据方式，可选：POST和GET

editData

[]

额外的数据提交到server

recreateForm

false

是否每次打开dialog都重新创建

addedrow

first

（新增行可用）设置新增行所在位置,可选first（首行）和 last（尾行）

topinfo

“”

在标题下方添加信息行内容

bottominfo

“”

在按钮下方添加信息行内容

saveicon

[true,”left”,”ui-icon-disk”].

定义保存按钮

closeicon

[true,”left”,”ui-icon-close”]

定义取消按钮

checkOnSubmit

false

（编辑数据可用）是否保持编辑过数据时显示确认框

closeAfterAdd

false

（新增行时可用）新增行数据后关闭dialog

clearAfterAdd

true

（新增行时可用）新增行数据后清除dialog数据

closeAfterEdit

false

（编辑行时可用）编辑行数据后关闭dialog

reloadAfterSubmit

true

提交数据后重新加载grid

closeOnEscape

false

是否可用“Esc”键关闭窗口

ajaxEditOptions

{}

Ajax属性，可覆盖默认ajax所有属性

viewPagerButtons

true

是否显示调整“上下”数据按钮

事件

afterclickPgButtons

（编辑数据时可用）点击调整“上下”数据按钮后触发

whichbutton：'prev' or 'next'；

formid：form id值；

rowid：当前行id；

afterComplete

新增或者更新数据完成后触发

 

response：server端返回数据；

postdata：发送到server端的数据；

formid：弹出框中form id值；

afterShowForm

显示dialog form数据后出发

formid：弹出框中form id值

afterSubmit

接收到server返回信息后触发

response：server端返回数据；

postdata：发送到server端的数据；

此事件要求返回数据格式为：

[success, message, new_id]

success：为true时继续处理，为false时显示message信息并停止后续处理。

new_id：新增行处理时用做新行id；

beforeCheckValues

校验单元格值前触发（colModel中editrules定义的规则）

postdata：发送到server端的数据；

formid：弹出框中form id值；

mode：当前模型（编辑或新增）

beforeInitData

初始化dialog新数据前触发

formid：弹出框中form id值；

beforeShowForm

显示dialog新数据前触发

formid：弹出框中form id值；

beforeSubmit

提交数据到server前触发

postdata：发送到server端的数据；

formid：弹出框中form id值；

此事件要求返回数据格式为：

[success, message]

success：为true时继续处理，为false时显示message信息并停止后续处理。

onclickPgButtons

（编辑数据时可用）点击调整“上下”数据按钮时触发

whichbutton：'prev' or 'next'；

formid：form id值；

rowid：当前行id；

onclickSubmit

点击“提交”按钮时触发

Params：editGridRow方法所有属性；

postdata：发送到server端的数据；

此事件要求返回数据格式为：

{}，扩展postdata；

onInitializeForm

首次创建dialog时触发

formid：弹出框中form id值；

onClose

关闭窗口事件

当关闭弹出框之前触发，即单击关闭关窗口或点击取消按钮。如返回true弹出框关闭，否则不关闭。

errorTextFormat

Ajax请求错误时触发

返回错误信息

serializeEditData

提交数据时序列化提交到server端数据

可以设置自定义数据
