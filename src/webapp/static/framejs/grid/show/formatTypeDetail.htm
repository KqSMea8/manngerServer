格式化、反格式说明：

数据格式化支持自定义，通过把formatter的值设置为自定义函数对数据进行格式化，同样通过设置unformat可以反格式化， 例如：数值123格式化后为$123,取值时只想获取123，此时就可在反格式化函数里进行处理。
new biz.grid({ ...
colModel: [ ...
{……
formatter:currencyFmatter, //调用格式化函数
unformat:unformatCurrency}, //调用反格式化函数
... ] ...
});
function currencyFmatter (cellvalue, options, rowObject){ return "$"+cellvalue; }
function unformatCurrency (cellvalue, options,cellObject){ return cellvalue.replace("$","");}
格式化函数参数说明：
cellvalue：要被格式化的值（格式化函数），单元格文本值（反格式化函数）；
options：对数据进行格式化时的相关参数，格式为：
{ rowId: rid, colModel: cm} ；
rowObject：行数据对象；
cellObject:格式化后的cell数据对象；
