[文件路径: grid/show/selfAdaption.html]
概述
展示grid自适应容器大小功能，设置grid容器大小为父容器的百分比；同时能指定固定值，不管父容器是否改变大小，grid大小始终不变；grid高宽默认为auto；

 var selfAdaptionObj = new biz.grid({
