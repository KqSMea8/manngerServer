说明

addJSONData

data

none

使用传来的data数据填充表格

addRowData

rowid、

data、

position、

srcrowid

true（增加数据成功）/false（增加数据失败）

根据参数插入一行新的数据。

rowid: 新行的id

data: 为新行的数据

position: 为新增行的位置

srcrowid: 为新增行的参考位置

delRowData

rowid

true（增加数据成功）/false（增加数据失败）

根据rowid删除行,不从服务器删除

getRowData

rowid/none

array{}

rowid：行id,none：参数为空时返回所有行数据,返回指定行的数据（非编辑后的值）

setRowData

rowid、

data、

cssprop

成功true否则false

更新行的值，rowid为行id。data值格式：{name1:value1,name2: value2…} name为colModel中名称；cssprop：如果是string则会使用addClass方法将其加入到行的css中，如果是array或者对象则会直接加到style属性中

addLocalRow

rowid、

data、

position、

srcrowid

true（增加数据成功）/false（增加数据失败）

 
参数同addRowData，与其区别：addLocalRow增加的数据能通过getAdded、getJsontosubmit获取，
且新增单元格有标示；。

setLocalRow

rowid、

data、

cssprop

成功true否则false

参数同setRowData，与其区别：setLocalRow设置的数据能通过getEdited、getJsontosubmit获取，且设置单元格有标示；

delLocalRow

rowid

true（增加数据成功）/false（增加数据失败）

参数同delRowData，与其区别：delLocalRow删除的数据能通过getDeled、getJsontosubmit获取。

bindKeys

}

grid 对象

自定义绑定事件处理

clearGridData

clearfooter

grid 对象

清除grid数据；

clearfooter：是否清除gridfooter部分，true清除，false不清除；

footerData

action、

data、

format

grid对象

设置或者取得底部数据。

action：“get”或者“set”，默认为“get”，如果为“get”返回值为name:value，name为colModel中名称。如果为“set”则值为name：value，name是colModel中的名称。format：默认为true，当为true时，在设置新值时会调用formatter格式化数值

getCell

rowid、

iCol

单元格内容

返回指定rowid，iCol的单元格内容。rowid：行id，iCol：可以是当前列在colModel中的位置索引也可以是name值。注意：在编辑行或者单元格时不能使用此方法，此时返回的并不是改变的值，而是原始值

getCol

colname、returntype、mathoperation

array[] or value

返回列的值（可返回平均值、总和、数量、列对象等）。colname：既可以是当前列在colModel中的位置索引也可以是name值
returntype：指定返回数据的类型，默认为false。当为false时，返回的数组中只包含列的值，当为true时返回数组是对象数组；mathoperation：可选值为'sum’, 'avg', 'count'

getDataIDs

none

array[]

返回当前grid里所有数据的id

getGridParam

name

mixed value

返回请求的参数信息

getInd

rowid,
rowcontent

mixed

当id=rowed且rowcontent设置为false时返回行序号；rowcontent为true时返回行数据，如果rowid没发现，返回false

remapColumns

permutation, updateCells, keepHeader

none

调整表格列的显示顺序,permutation为当前列的顺序，假如值是[1,0,2]，那么第一列就会在第二位显示。如果updateCells为ture则是对单元格数据进行重新排序，如果keepHeader为true则对header数据显示位置进行调整

resetSelection

none

grid对象

选择或者反选行数据，在多选模式下也同样起作用

hideCol

Colnameor/[colnames]

grid对象

如果参数为一个列名则隐藏此列，如果给定的是数组则隐藏指定的所有列

showCol

colname

grid对象

显示列。colname可以是数组[“name1”,”name2”],但是name1或者name2必须是colModel中的name

setCaption

caption

grid对象

设置表格的标题

setSelection

rowid,onselectrow

grid对象

选择或反选指定行。如果onselectrow为ture则会触发事件onSelectRow，onselectrow默认为ture

setCell

rowid、

colname、

data、

class、

properties

grid对象

改变单元格的值。rowid：当前行id；colname：列名称，也可以是列的位置索引，从0开始；data：改变单元格的内容，如果为空则不更新；class：如果是string则会使用addClass方法将其加入到单元格的css中，如果是array则会直接加到style属性中；properties：设置单元格属性

setGridParam

object

grid对象

设置grid的参数。有些参数的修改必须要重新加载grid才可以生效，这个方法可以覆盖事件

setGridHeight

new_height

grid对象

动态改变grid的高度，只能对单元格的高度进行设置而不能对表格的高度进行动态修改。new_height：可以是象素值，百分比或者"auto"

setGridWidth

new_width、shrink

grid对象

动态改变表格的宽度。new_width:表格宽度，象素值；shrink：true或者false，作用同shrinkToFit

setLabel

colname、

data、

class、

properties

grid对象

给指定列设置一个新的显示名称。colname：列名称，也可以是列的位置索引，从0开始；data：列显示名称，如果为空则不修改；class：如果是string则会使用addClass方法将其加入到单元格的css中，如果是array则会直接加到style属性中；properties：设置label的属性

trigger(“reloadGrid”)

none

none

重新加载当前表格，也会向服务器发起新的请求
