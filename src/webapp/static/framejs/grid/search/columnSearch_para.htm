类型

属性

默认值

说明

基本属性

autosearch

true

是否自动查找：input元素当按“Enter”、select元素当选项改变时自动查找。

beforeSearch

null

查找grid之前触发。如返回值为true，将不会触发grid查找数据，用户可以自行组件参数并且触发grid；如返回其他值，继续查找grid。

afterSearch

null

查找数据之后触发

beforeClear

null

清除查询框内容之前触发。如返回值为true，将不会触发grid查找数据，用户可以自行组件参数并且触发grid；如返回其他值，继续查找grid。

afterClear

null

清除查询框内容之后触发

searchOnEnter

true

是否通过“Enter”触发查找，如为true参考autosearch属性说明，为false时每输入一个字符进行数据查找。

stringResult

false

设置提交查找数据信息方式，为false时通过key:value方式；True为通过searchGrid提交。

groupOp

AND

设置分组方式，可选AND 、 OR

defaultSearch

bw

可选内容参见查找属性：sopt
