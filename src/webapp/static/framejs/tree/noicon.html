示例

    • 1、setting 配置信息说明

      • 此Demo利用 Function 设置了使父节点不显示图标的规则◦ 是否显示节点图标请设置 setting.view.showIcon 属性

    • 2、treeNode 节点数据说明

      • 是否显示图标，不需要 treeNode 节点数据提供特殊设置，但如果用户需要根据不同节点动态设置，可以对节点数据增加特殊属性，用于判别
         源代码说明

	var setting_noicon = {
	        id:"#treeDemo_noicon",//容器id
            nodes:zNodes,  //数据节点指定
			view: {
				showIcon: showIconForTree //显示图标设置
			},
			data: {
				simpleData: {
					enable: true
				}
			}
	};
	function showIconForTree(treeId, treeNode) {
			return !treeNode.isParent; //只有叶子节点设置图标
	};
