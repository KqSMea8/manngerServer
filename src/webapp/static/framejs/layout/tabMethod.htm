方法

参数

说明

destroy

无

选项卡销毁

disable

无

整个选项卡不可用

enable

无

整个选项卡可用

option

optionName、value

获取/设置属性. optionName:属性名；value：属性值；如果参数为空则为获取属性值。

add

url 、label 、index

添加选项卡。url: tab标签内容设置url；label：标题；index：标签序号；

remove

index

删除选项卡。index：标签序号；

select

index

选择一个选项卡标签。index：标签序号；

load

index

重载一个ajax选项卡的内容，这个一直载入远程内容。index：标签序号；

url

index 、url

改变一个标签页的url。index：标签序号；url：url地址。

abort

无

中止所有运行在tab标签上的ajax请求或动画

rotate

ms、continuing

自动翻滚选项卡标签。ms:翻滚间隔时间；continuing：当一个标签选择后是否继续翻滚。
